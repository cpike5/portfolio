Card Features 
Common Features:
  • Gradient background (primary color with transparency)
  • Border: 1px solid rgba(183, 65, 254, 0.2)
  • Border radius: 8px
  • Hover elevation effect (translateY)
  • Top gradient accent (appears on hover)
  • Smooth transitions (0.3s ease)

Project Card:
  • 2rem padding
  • Flex column layout
  • Image/placeholder at top
  • Link at bottom (margin-top: auto)
  • Hover: translateY(-5px), title shifts right

Skill Card:
  • Centered text
  • 1rem vertical, 1.5rem horizontal padding
  • 2px top accent on hover
  • Hover: translateY(-3px)

Work Card:
  • 2rem padding
  • Bottom border separator
  • Hover: background tint, translateY(-4px)

Implementation Notes

Design Pattern:
• All cards share gradient background and border styling
• Top gradient accent line (3px for project, 2px for skill)
• Hover effects use GPU-accelerated transforms
• Consistent border radius and spacing

Responsive:
• Project cards: Grid auto-fit, min 300px
• Skill cards: 4 columns → 2 columns on mobile
• Work cards: Stack vertically, reduced padding
• Mobile padding: 1.5rem → 1.25rem

Accessibility:
• Semantic HTML (article, h3, p, a)
• Proper alt text on images
• Focus states visible
• Links have descriptive text

Performance:
• Transform and opacity for animations
• No layout thrashing
• Lazy load images when implemented
