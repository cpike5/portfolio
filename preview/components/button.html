Button

Primary button component with outlined style, hover fill effect, and loading/disabled states. Uses small-caps typography and smooth transitions.

HTML Structure 
<!-- Default Button -->
<button type="submit" class="form-submit">
    <span class="submit-text">Submit</span>
    <span class="submit-loader"></span>
</button>

<!-- Loading State -->
<button type="submit" class="form-submit loading">
    <span class="submit-text">Submit</span>
    <span class="submit-loader"></span>
</button>

<!-- Disabled State -->
<button type="submit" class="form-submit" disabled>
    <span class="submit-text">Submit</span>
    <span class="submit-loader"></span>
</button>
CSS Classes & Styling 
Classes:
  .form-submit     - Button base styling
  .submit-text     - Button text (hidden when loading)
  .submit-loader   - Loading spinner (shown when loading)
  .loading         - Loading state modifier

Button Styling:
  • Padding: 1rem 2.5rem
  • Background: transparent
  • Border: 2px solid var(--color-primary)
  • Border-radius: 4px
  • Color: var(--color-primary)
  • Font-variant: small-caps
  • Letter-spacing: 0.05em
  • Min-width: 200px
  • Position: relative

Hover State:
  • Background: var(--color-primary)
  • Color: var(--color-background)
  • Transform: translateY(-2px)

Loading State:
  • .submit-text opacity: 0
  • .submit-loader display: block
  • Spinner animation: 0.8s linear infinite
  • No hover effects

Disabled State:
  • Opacity: 0.6
  • Cursor: not-allowed
  • No hover effects

JavaScript (Loading State) 
// Add loading state
button.classList.add('loading');
button.disabled = true;

// Remove loading state
button.classList.remove('loading');
button.disabled = false;

Implementation Notes

Design Pattern:
• Outlined style (transparent background, border)
• Fills on hover for visual feedback
• Small-caps for consistent typography
• Loading spinner replaces text

Animation:
• Hover lift: translateY(-2px)
• Active press: translateY(0)
• Spinner rotation: 360deg in 0.8s
• All transitions: 0.3s ease

States:
• Default: outlined with primary border
• Hover: filled with primary color
• Loading: spinner visible, text hidden, no hover
• Disabled: reduced opacity, no interaction

Accessibility:
• Proper button element
• Type attribute (button/submit)
• Disabled attribute for disabled state
• Focus states visible
• ARIA attributes for loading state
