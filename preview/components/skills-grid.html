CSS Classes & Styling 
Classes:
  .skill-group         - Group container
  .skill-group-title   - Group heading
  .skills-grid         - Grid container
  .skill-item          - Individual skill card

Grid Layout:
  • Display: grid
  • Template: repeat(auto-fit, minmax(140px, 1fr))
  • Gap: 1rem
  • Max-width: 800px (centered)

Skill Item:
  • Flexbox centered
  • Padding: 1rem 1.5rem
  • Border: 1px solid rgba(183, 65, 254, 0.2)
  • Border-radius: 8px
  • Background: gradient (primary with low opacity)
  • Top accent: 2px gradient line
  • Font-size: 0.95rem

Hover Effects:
  • Border color: solid primary
  • Background: intensified gradient
  • Transform: translateY(-3px)
  • Top accent opacity: 0.6

Group Styling:
  • Margin-bottom: 3rem (last: 0)
  • Title: small-caps, primary color
  • Title margin-bottom: 1.5rem

Responsive:
  • Desktop: auto-fit (typically 4-5 columns)
  • Tablet (≤768px): 2 columns
  • Mobile (≤480px): 2 columns
  • Reduced padding and gaps

Blazor Component Props 
Proposed Props:

SkillGroups (List<SkillGroup>) [Required]
  - Collection of skill groups

SkillGroup Model:
  - Title: string (group name)
  - Skills: List<string> (skill names)

MaxWidth (string)
  - Grid max-width (default: "800px")

Columns (int)
  - Desktop columns (default: auto-fit)

MobileColumns (int)
  - Mobile columns (default: 2)

CssClass (string)
  - Additional CSS classes

Implementation Notes

Grid Behavior:
• Auto-fit adjusts columns based on available space
• Minimum column width: 140px
• Expands to fill available space (1fr)
• Responsive without media queries (for column count)

Spacing:
• Grid gap: 1rem
• Group margin-bottom: 3rem
• Title margin-bottom: 1.5rem
• Reduces on mobile for tighter layout

Interaction:
• Hover lift effect on skill items
• Top gradient accent appears
• Border color intensifies
• Smooth 0.3s transitions

Content Organization:
• Group related skills together
• Use descriptive group titles
• Order groups by importance
• Keep skill names concise

Accessibility:
• Semantic heading hierarchy (h3)
• Logical reading order
• Sufficient color contrast
• Clear visual grouping
