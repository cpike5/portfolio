PageSection

Container component for page sections with consistent spacing, centering, and optional fade-in animations. Provides configurable max-width and responsive padding.

Phase 1 Layout Container

Layout Variants

About Section (max-width: 700px)

About

This section uses a 700px max-width content container, ideal for readable text content and biographical information.

Skills Section (max-width: 900px)

Skills

This section uses a 900px max-width, providing more room for skill grids and multi-column layouts.

Projects Section (max-width: 1000px)

Projects

This section uses a 1000px max-width for wider layouts like project grids and card displays.

Full-Height Section

Full viewport height with centered content

Centered Content

Sections use flexbox to vertically and horizontally center content. The min-height can be set to 100vh for full-page sections.

Blazor Component Props 
Proposed Props:

Id (string)
  - Section identifier for navigation anchors

MaxWidth (string)
  - Content max-width: "700px" | "800px" | "900px" | "1000px"

EnableFadeIn (bool)
  - Toggle scroll-triggered fade-in animation

FullHeight (bool)
  - Use min-height: 100vh

CssClass (string)
  - Additional CSS classes

ChildContent (RenderFragment)
  - Section content

Implementation Notes

Layout:
• Flexbox centered (justify-content: center, align-items: center)
• Full-height sections: min-height: 100vh
• Responsive padding: 4rem → 3rem → 2.5rem
• Content containers limit width for readability

Animation:
• Intersection Observer triggers fade-in
• Threshold: 0.1 (10% visible)
• Transforms from translateY(30px) to translateY(0)
• Opacity animates from 0 to 1

Max-Width Guidelines:
• 600px: Forms, narrow content
• 700px: Text-heavy sections (About)
• 800px: Mixed content (Work)
• 900px: Grid layouts (Skills)
• 1000px: Card grids (Projects)
