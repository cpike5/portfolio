CSS Classes & Styling 
Classes:
  .hero              - Hero section container
  .hero-title        - Ultra-light title text
  .scroll-indicator  - Scroll arrow link
  .arrow-down        - CSS-drawn arrow

Hero Container:
  • Display: flex
  • Center: justify-content & align-items
  • Min-height: calc(100vh - 6rem)
  • Text-align: center
  • Position: relative (for absolute arrow)

Title Styling:
  • Font-family: var(--font-family-display)
  • Font-size: 5rem → 3.5rem → 2.5rem
  • Font-weight: 100 (ultra-light)
  • Color: var(--color-primary)
  • Letter-spacing: 0.05em

Scroll Arrow:
  • Position: absolute bottom 2rem
  • 24px × 24px CSS-drawn arrow
  • 2px borders forming 45deg rotated square
  • Bounce animation (2s infinite)
  • Hover: opacity & color change

Animation Keyframes:
  @keyframes bounce {
    0%, 20%, 50%, 80%, 100% { translateY(0) }
    40% { translateY(-10px) }
    60% { translateY(-5px) }
  }

Styling Reference:
  styles.css:302-355
Blazor Component Props 
Proposed Props:

Title (string) [Required]
  - Hero title text (default: "cpike.ca")

ScrollTarget (string)
  - Scroll target anchor (default: "#about")

ShowScrollIndicator (bool)
  - Show/hide scroll arrow (default: true)

MinHeight (string)
  - Custom min-height (default: "calc(100vh - 6rem)")

CssClass (string)
  - Additional CSS classes

Implementation Notes

Layout:
• Full viewport height minus navbar
• Flexbox centered content
• Relative positioning for absolute arrow
• Mobile-responsive height adjustments

Typography:
• Ultra-light weight (100) for dramatic effect
• Large font size with responsive scaling
• Letter-spacing for visual breathing room
• Primary color maintains brand consistency

Animation:
• Scroll arrow bounces continuously
• Smooth 2-second loop
• Opacity changes on hover
• Respects prefers-reduced-motion

Interaction:
• Clicking arrow smoothly scrolls to next section
• Hover changes arrow color and opacity
• Touch-friendly target size (24px + padding)
