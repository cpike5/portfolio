HTML Structure 
<nav class="navbar">
    <ul class="nav-links">
        <li><a href="#home" class="active">Home</a></li>
        <li><a href="#about">About</a></li>
        <li><a href="#skills">Skills</a></li>
        <li><a href="#projects">Projects</a></li>
        <li><a href="#work">Work</a></li>
        <li><a href="#contact">Contact</a></li>
    </ul>
</nav>
CSS Classes 
Key Classes:
  .navbar           - Sticky navigation container
  .nav-links        - Flexbox list container
  .nav-links a      - Navigation link styling
  .active           - Active state indicator

Styling Reference:
  styles.css:230-296

Features:
  • Sticky positioning (top: 0)
  • Centered horizontal layout (gap: 3rem)
  • Small-caps font variant
  • Animated underline on hover
  • Gradient separator below navbar
  • Active state with full underline
Blazor Component Props 
Proposed Props:

NavItems (List<NavItem>)
  - Text: string (display text)
  - Href: string (link/route)
  - IsActive: bool (active state)

CssClass (string)
  - Additional CSS classes

OnNavigate (EventCallback<string>)
  - Navigation event handler

Implementation Notes

Behavior:
• Uses Intersection Observer to track active section
• Smooth scroll on link click
• Automatically updates active state based on scroll position

Accessibility:
• Semantic <nav> element
• Proper list structure
• Keyboard navigable
• Focus states visible

Responsive:
• Hidden on mobile (<768px)
• MobileNavigation shown instead
• Reduce gap on smaller screens if needed
