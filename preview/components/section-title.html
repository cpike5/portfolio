CSS Classes & Styling 
Classes:
  .section-title         - Base section title styling
  .animate-underline     - Triggers underline animation

CSS Custom Properties:
  --font-family-display       (Inter)
  --font-size-section-title   (2.5rem → 2rem → 1.75rem)
  --font-weight-section-title (300)
  --color-primary             (#b741fe)
  --color-primary-hover       (#d680ff)

Typography:
  • Small-caps variant
  • Letter-spacing: 0.08em
  • Text-transform: lowercase
  • Inline-block display

Underline:
  • Positioned -8px below text
  • Height: 2px
  • Gradient: primary → primary-hover
  • Animates width: 0 → 100%
  • Transition: 0.8s cubic-bezier(0.4, 0, 0.2, 1)

JavaScript (Intersection Observer) 
const sectionTitles = document.querySelectorAll('.section-title');

const titleObserver = new IntersectionObserver((entries) => {
    entries.forEach(entry => {
        if (entry.isIntersecting) {
            entry.target.classList.add('animate-underline');
        }
    });
}, {
    root: null,
    rootMargin: '0px',
    threshold: 0.5  // Trigger when 50% visible
});

sectionTitles.forEach(title => titleObserver.observe(title));

Implementation Notes

Typography:
• Font-weight: 300 (light)
• Small-caps with lowercase text
• Letter-spacing for visual breathing room
• Display: inline-block (for ::after positioning)

Animation:
• Triggers at 50% visibility threshold
• Smooth cubic-bezier easing
• Gradient underline adds visual interest
• Only animates once (doesn't reverse on scroll out)

Responsive:
• 2.5rem (desktop)
• 2rem (tablet ≤768px)
• 1.75rem (mobile ≤480px)

Accessibility:
• Proper heading hierarchy (h2 by default)
• Sufficient color contrast
• Animation respects prefers-reduced-motion
